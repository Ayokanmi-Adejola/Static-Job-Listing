How to Test

  1. View the application in the iframe above
  2. Try clicking on different filter tags (Frontend, Senior, JavaScript, etc.)
  3. Notice how the filter bar appears with active filters
  4. Test the "Clear" button to remove all filters
  5. Resize the browser window to test responsive design
  6. Check hover effects on job titles and filter tags

Browser Console

Open browser developer tools (F12) and check the console for any errors or debug messages.

You should see messages like:

  • "DOM loaded, initializing app..."
  • "Loading jobs data..." or "Using fallback data..."
